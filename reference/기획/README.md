## 현재 모델 — 3-Agent: 소비 분석 · 지킴·성장 · 취향·추천 (PRD v9)

> ① 소비 분석 · ② 지킴·성장(금액+게이미피케이션) · ③ 취향·추천(리포트+맞춤 금융상품 **정보성 비교** 추천). 금융상품은 정보성 비교만(판매·중개 없음, `가입은 각 사로`).

```text
사용 한도 = 기준 소비 × (1 − 절약 강도)   이번 달 써도 되는 돈
지킬 돈   = 기준 소비 × 절약 강도          이번 달 지키기로 한 돈
지킨 돈   = max(지킬 돈 − max(누적 소비 − 사용 한도, 0), 0)
```

한도를 넘긴 만큼만 깎인다. 강도 100%를 고르면 사용 한도가 0원이 되어 카테고리 전체 절감(v6)과 같아지므로, **전체 절감은 부분 절감의 특수한 경우**다.

금액 모델 위에 캐릭터 여정·성장 층을 얹는다. 사용자의 목표는 별도의 물건·목적지가 아니라 `아낄 수 있는 돈을 아끼는 행위` 자체이며, 이를 캐릭터의 성장으로 동기 치환한다. 성장 층은 `월말 확정 지킨 돈`을 먹으며 월 경계를 넘어 누적되고 리셋·퇴화하지 않는다. 목표(물건)·보상·육성 오브젝트·쿠폰·참기 세션은 폐기했고, OS 위젯은 코어 가설과 무관해 Later로 이동했다.
