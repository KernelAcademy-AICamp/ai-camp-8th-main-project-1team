## 1. v1 대비 무엇이 강해졌나

- **P1**: 지불고통 이론(Prelec & Simester 2001 등) + 정량 사례(카드 결제일 미스매치: 전달 65만 소비 vs 137만 청구, 머니투데이) 추가.
- **P4**: 시발비용 정량(대학내일 20대 62.5% 경험) + 26세 신입 주 30만 사례.
- **P4·P5 연결**: 25세 사회초년(월 400만) 보상소비 → 저축 실패, 금감원 재무상담 인용.
- **P3**: 거지방 규칙 상세화 + 뉴워커 827명(88.4% 생활 빠듯, 81% 지출 축소 노력) 사회 규모.
- **P6 승격 (★★★★☆ → ★★★★★)**: 클리앙 3개 스레드에서 "느리고 답답, 피드백 없음, 데이터 꼬임, 누락" 확인.

## 2. 검증에서 기각된 클레임 (PRD 인용 금지)

- "직장인 85.5%가 월평균 20만원 홧김비용" (아시아경제) — 원출처 재검증 실패
- "20~30대 59.1% 스트레스 소비 정당화" — 완전 기각
- "카드 결제 시점 놀람이 대표적 반응" (머니투데이) — 헤드라인 수사법
- **"무지출 챌린지 = 가계부/금융 앱 인증"** (나무위키) — **완전 기각. 실제는 커뮤니티 채팅방 인증 중심** ← 사적 앱 기록 욕구는 별도 검증 필요
- "장시간 노동 = 저축 실패 주요 원인" — 단일 케이스 일반화 (개인 사례는 인용 가능)

**교훈**: 통계는 원출처 재확인 필수 · 개별 사례는 인용 가능하나 일반화 금지.

## 3. 추가 시사점

- **P6 MVP 함의**: 뱅샐이 걸어간 길(자동 밀되 오류 방치 → 데이터 무결성 문제 → 부분 이탈)을 피해야. 오류를 즉시 눈치채고·수정 가능하고·학습 반영되는 3단계 UX가 핵심.
- **포지셔닝 힌트**: "가계부는 절약 도구가 아니라 돈을 소중히 여기게 만드는 의식 행위" → "당신의 돈에 이름을 붙여드립니다".
- **새 축 P4-6**: 관계 소비(연인·친구 만남 페이 압박) 추가 검토.
